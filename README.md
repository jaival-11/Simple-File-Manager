## Features

* **No bloat:** Simple fast file manager without any unnecessary features.
* **Fully Open-Source:** Transparent development with zero tracking, built entirely using modern FOSS principles.
* **Modern & Lightweight UI**: Built with a clean, Material Design aesthetic with monet themeing.
* **Privacy respecting**: Entire app works offline and never connects to internet. No data is collected by the developer.

## Bug Reports & Feature Suggestions

Encountered a bug or have a brilliant idea to make Simple File Manager even better?

* **Found a bug?**   

Open a [Bug Report](https://github.com/jaival-11/Simple-File-Manager/issues/new?template=.github%2fISSUE_TEMPLATE%2fbug_report.yml) using the provided template. Please include necessary and correct details so it can be squashed quickly!  

* **Have an idea?**   

Open a [Feature Request](https://github.com/jaival-11/Simple-File-Manager/issues/new?template=.github%2fISSUE_TEMPLATE%2ffeature_request.yml) using the provided template and describe how it would improve the app. Not all features will be added to maintain the app lightweight as intended.

### Got any questions?

Feel free to open an [Issue](https://github.com/jaival-11/Simple-File-Manager/issues/new/choose)

*You can also contact me directly via details given at the top, though [Github Issues](https://github.com/jaival-11/Simple-File-Manager/issues) is preferred.*
